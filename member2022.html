Live Streaming Platform - Member

UI / Web Design

2018, 2022

Overview:
Designed user interfaces for a live streaming platform's member-facing application across mobile (iOS/Android, 2018) and web (2022, unreleased). The project focused on creating an engaging viewing experience for audience members to discover content and interact with streams.

Contributions:
• Designed user flows and information architecture for member features
• Developed visual design system for mobile app (2018)
• Designed and coded responsive web interface update using HTML/CSS (2022)

Design Approach:
Focused on creating a modern, clean interface with intuitive navigation that enhances content discovery and viewing experience. Design prioritized seamless interaction flows and maintained consistent visual language across platforms to improve usability and member engagement.

Tools:
Figma, HTML, CSS
