Wedding

Logo / Graphic / Web Design

2023

Overview:
Designed a complete wedding suite for a couple, including logo, print invitations, and an interactive digital invitation website.

Contributions:
• Developed wedding logo and visual identity
• Designed print invitation suite
• Created and coded responsive digital invitation website using HTML, CSS, and JavaScript

Design Approach:
The design concept drew inspiration from the couple's long-distance relationship, using soft cloud imagery—as seen through an airplane window—to symbolize their journeys toward reunion. A blush pink color palette evokes warmth and hope, while modern calligraphy adds a playful, romantic touch that reflects the joy of their connection.

Tools:
Adobe Illustrator, Photoshop, Figma, HTML, CSS, JavaScript
